Keybit8を動作させるには、ProMicroへのファームウェアの書き込みが必要です。

書き込みについてはこの後の[5章](#5-正規ファームウェアの書き込み)で製品版ファームウェアの書き込みについて詳しく説明しています。

ここからはんだ付けと組み立て作業を行なっていきます。  
はんだ付けパーツは全て基板の表面から差し込んで裏面にはんだ付けをしていきます。  

下記画像が基板（表面）です。  
![04](images/_MG_8242.jpg)  

下記画像が基板（裏面）です。  
![05](images/_MG_8257-2.jpg)  

<a id="anchor3-1"></a>
### 3-1. はんだ付けの前に
はんだごての"こて先"は竹槍の先端のような形状のものだと平面にはんだを乗せたり、余分なはんだを"すくい取る"事が出来るので使いやすいです。  
![06](images/kb39_027.jpg)

<a id="anchor3-2"></a>
### 3-2. 抵抗器の取り付け
それでは抵抗器を取り付けていきます。  
画像を参考にし、基板に記載されている抵抗値と抵抗器の色帯を確認しながら、正しい抵抗器を取り付けてください。    

>**⚡️注意**:抵抗器の取り付け箇所を間違えると動作しなくなるので、1個ずつ確認しながら取り付けを行ってください。  
取り付け箇所を間違えた場合は、はんだ吸い取り機、又ははんだ吸い取り線ではんだを除去して部品を取り外してください。

まずR12の抵抗器（赤黒赤金の色帯）を取り付けて、はんだ付けをします。  
![07](images/_MG_8276.jpg)   

画像のように抵抗器のピンを折り曲げて基板のスルーホールに差し込みます。  
![08](images/_MG_8258.jpg)  

基板裏面から抵抗器のピンを折り曲げて抵抗器が外れないように固定すると、はんだ付けしやすいです。  
![09](images/_MG_8277.jpg)  
  
はんだ付けをした後は余分なピンをニッパーでカットします。  
ピンをカットした後はんだを再加熱すると、きれいに仕上がります。（気になる方は実施してください。）  
![10](images/_MG_8285-2.jpg)

次にR10とR11の抵抗器（赤紫赤金の色帯）を1個ずつ取り付けて、はんだ付けをしてください。   
![11](images/_MG_8295.jpg) 

最後にR1〜R9までの抵抗器（茶灰茶金の色帯）を同様に1個ずつ取り付けて、はんだ付けをしてください。  
全部で12本の抵抗器を取り付けます。  
取り付けた抵抗器が抵抗値と一致しているか最後に確認をしてください。  
![12](images/_MG_8303.jpg)

<a id="anchor3-3"></a>
### 3-3. ダイオードの取り付け
次にダイオードを取り付けていきます。ダイオードには取り付ける方向があるので注意してください。  
ダイオードは全部で8本取り付けます。  
抵抗器と同じようにピンを折り曲げてからスルーホールに差し込みます。  
![12](images/_MG_8309.jpg)

ダイオードの黒いライン（極性マーク）の向きを基板上の白いラインと合わせて取り付けます。
>**⚡️注意**:ダイオードの向きは8個全て同じ向きで取り付けます。  
はんだ付けをする前に極性マークの向きと、他のダイオードと同じ向きになっているか、確認してからはんだ付けしてください。   

![13](images/_MG_8311-3.jpg)

抵抗器と同様に差し込んだ後にピンを折り曲げて固定すると、はんだ付けしやすいです。  
![14](images/_MG_8312-2.jpg)

はんだ付け後  
![15](images/_MG_8315.jpg)

<a id="anchor3-4"></a>
### 3-4. トランジスタアレイの取り付け
トランジスタアレイを取り付けていきます。  
![16](images/_MG_8464.jpg)  

トランジスタアレイには半円形の切り欠きがあります。  
基板上の半円形マークと向きを合わせて差し込みます。  
![16](images/_MG_8478.jpg)  

トランジスタアレイを差し込んだ状態で、余ったピンをカットしてからはんだ付けすると、きれいに仕上がります。  
>**⚡️注意**:トランジスタアレイが浮いた状態ではんだ付けをしてしまうと修正が難しいので、はんだ付けをする前にしっかりと差し込まれた状態か確認してからはんだ付けをしてください。  
（難しい場合はマスキングテープで固定をすると作業しやすいです。）    

![17](images/_MG_8473.jpg)

<a id="anchor3-5"></a>
### 3-5. LEDの取り付け
LEDを取り付けていきます。  
LEDにも極性があり、下記画像のようにLEDの切り欠き（平らな部分）と基板の円形マークの切り欠き部分に合わせて取り付けてください。  
LEDには赤LED（透明レンズ）が8本と緑LED（緑レンズ）が1本あります。  
緑LEDの取り付け箇所には「GREEN」の表記があるので間違えないように注意してください。  
![20](images/_MG_8339.jpg)

LEDの切り欠き部分を基板のシルクの円形の切り欠き部分に合わせて差し込みます。  
![18](images/_MG_8341-2.jpg)

ピンが短い方が切り欠き方向になります。  
![19](images/_MG_8345-2.jpg)

LEDも同様にはんだ付けが難しい場合はマスキングテープなどで固定してからはんだ付けしてください。  
はんだ付け後は余分なピンをカットしてください。  
![20](images/_MG_8349-2.jpg)

取り付け後イメージ  
![21](images/_MG_8392-4.jpg)

<a id="anchor3-6"></a>
### 3-6. ProMicro・ピンヘッダ・ピンソケットの取り付け

後から差し込みづらくならないようにProMicro・ピンヘッダ・ピンソケットを全て基板に取り付けた状態ではんだ付けを行ってください。  
ピンヘッダーをピンソケットに差し込むにはコツが必要です。  
ProMicro・ピンヘッダ・ピンソケットを全て基板に軽く差し込んだ状態でProMicroを押さえながら差し込むと、しっかり奥まで差し込むことができます。  
※ピンヘッダとピンソケットは、はんだ付け後も着脱可能な状態が正常です。  
![22](images/_MG_8367-3.jpg)  

ProMicroのはんだ付けをする際はProMicroの基板をマスキングテープで保護してからはんだ付けをしてください。  
はんだカスがProMicroの基板部分に付着して故障の原因になることがあります。
>⚠️ピンの数が多いパーツはまず1〜2箇所はんだ付けを行い、目視でパーツが浮いていないことを確認してから残りのはんだ付けも完了させます。

![23](images/_MG_8440-2.jpg)

<a id="anchor3-7"></a>
### 3-7. タクトスイッチ・圧電スピーカーの取り付け

タクトスイッチの取り付けを行います。  
![24](images/_MG_8379.jpg)  

タクトスイッチを差し込んだ後に、余ったピンをニッパーでカットしてから、はんだ付けをするときれいに仕上がります。  
![25](images/_MG_8487.jpg)

仕上がりイメージ  
![26](images/_MG_8388.jpg)

次に圧電スピーカーの取り付けを行います。  
余分なピンをカットしてから基板に差し込みます。  
差し込んだ後にピンを折り曲げて固定してから、はんだ付けを行います。  
![27](images/_MG_8390.jpg)

はんだ付け後、余分なピンをカットします。  
気になる方は、はんだを再加熱してきれいに仕上げてください。  
![28](images/_MG_8400.jpg)

<a id="anchor3-8"></a>
### 3-8. キースイッチの取り付け

キースイッチは8個全てを差し込んでから、はんだ付けをすると安定して取り付けられます。  
![29](images/_MG_8411.jpg)

ピンは折れ曲がりやすいので、8個全て基板に当たって折れ曲がっていないか確認をしてください。  
![30](images/_MG_8413.jpg)

スルーホールからはみ出した余分なピンはカットしてからはんだ付けをすると、きれいに仕上がります。  
![31](images/_MG_8415.jpg)

## 5. 正規ファームウェアの書き込み

それでは最後に正規のファームウェアをProMicroに書き込みましょう。  

Keybit8は[Remap](https://remap-keys.app/)に対応しているので、特別なソフトをインストールしなくても、ブラウザからファームウェアの書き込みとキーマップの変更が可能です。  
Remapの使用方法について詳しく知りたい方は[自キ温泉ガイドのサリチル酸](https://salicylic-acid3.hatenablog.com/entry/remap-manual)の記事をご覧ください。

Remapを使用するために[RemapカタログのKeybit8ファームウェアページ](https://remap-keys.app/catalog/LeRwa1B3uvwBCAaavuI7)を開き、Keybit8_viaのファームウェアを書き込みます。  
※上記のリンクからアクセスした場合は手順3から始めてください。

1.[Remap](https://remap-keys.app/)にアクセスしてトップページの「キーボードを探す」を選択します。  
![34](images/SS1.jpg)   

2.「Keybit8」を検索して選択します。  
![35](images/SS2.jpg) 

3.「ファームウェア」のタブを選択し、Keybit8をUSBケーブルでデバイスに接続してから「書き込み」を選択します。  
![36](images/SS3.jpg) 
  
4.再度「書き込み」を選択してからKeybit8のポートを選択します。  
USBケーブルを接続しているのにキーボードが認識されない場合は、はんだ付けしたリセットスイッチ（タクトスイッチ）を素早く2回押すと認識されます。  
Keybit8のポートを選択し、「接続」を押すとファームウェアの書き込みが開始されます。  
![37](images/SS4.jpg) 

これで書き込みは完了です。  

**ご自身でファームウェアをビルドする場合**   
Keybit8の最新ファームウェアは[Githubのリポジトリ](https://github.com/Yowkees/qmk_firmware/tree/master/keyboards/keybit8)にて公開しています。  
このkeybit8フォルダをお手持ちのQMK_Firmware/keyboardsへコピーして自由にキーマップ等を編集してmakeしてください。

## 6. キーマップの変更方法 
   [Remap](https://remap-keys.app/configure)で簡単にキーマップの確認・変更・書き込みが可能です。  
   [Remap](https://remap-keys.app/)のトップページから「キーボードをカスタマイズ」を選択し、「+キーボード」からKeybit8を選択します。

   ①変更したいキーのレイヤーを選択します。  
   ![38](images/SS5.jpg)   

   ②使用したいキーを変更したいキーへドラッグ&ドロップするだけで簡単に変更できます。  
   ![39](images/SS6.jpg)   

   その他の変更方法として、変更したいキーをクリックすると下記画像のようにポップアップが表示されるので、「キーコード」の入力欄に使用したいキーを入力します。  
   又は「キーコード」の入力欄を選択すると、使用できるキーの一覧が表示されるので、一覧から使用したいキーを選択することも可能です。  
   「Ctrl＋C」等のショートカットキーを1キーで入力したい場合は下記画像のように修飾キーを選択します。  
   ![40](images/SS7.jpg)   

   「ホールド/タップ」タブを選択すると、キーを短く押したとき（タップ）と押し続けたとき（ホールド）で、それぞれ異なるキー入力を設定できます。  
   ![41](images/SS8.jpg)  

   「カスタム」のタブでは使用したいキーが見つからない場合等に16進数でキーを割り当てることが可能です。  
   分からない場合はAIに「〇〇（入力したいキーや機能）のVIAキーコードの16進数は？」等と質問すると回答してくれるので、コピペして設定することも可能です。  
   「ラベル」には自動で名前が付けられますが、自分で名前をつけることもできます。  
   ![42](images/SS9.jpg)   

   Keybit8はキーボードとして使用する方法以外に、8個のキーと9個のLEDと圧電スピーカーを使用したゲームで遊ぶことができます。  
   左側4つのキー(1.2.3.4)を同時に長押しすることで「キー入力モード」と「ゲームモード」を切り替えられます。 （キーの番号は下記画像を参照） 

   ゲームモードのとき、次のキーの組み合わせを同時に押すと、それぞれのゲームで遊ぶことができます  
   1 + 3 を同時に押す → 「オルガン」  
   1 + 5 を同時に押す → 「ルーレットゲーム  
   1 + 7 を同時に押す → 「タイピングゲーム」  

   以上の3つのゲームが正規のファームウェアに書き込まれています。   

   各ゲームを終了するには右側4つのキー(5.6.7.8)を同時に長押ししてください。※各ゲームを終了しても引き続きゲームモードの状態は維持されています。  
   キー入力モードに切り替えたい場合は左側4つのキー(1.2.3.4)を同時に長押ししてください。  
   ![43](images/SS10.jpg)

## 7. Arduino IDEの使用方法

   Keybit8は[Arduino IDE](https://www.arduino.cc/en/software)を使用して自分でプログラムを書くことで、ゲームなどを作成して楽しむことができます。

   [Arduino IDE](https://www.arduino.cc/en/software)の使用方法  
   
   1.Arduino IDEをインストールする。  
   ![44](images/SS11.jpg)

   2.Arduino IDEを開いてボードの設定をする。（ツール → ボード → Arduino AVR Boards → Arduino Leonardo を選ぶ）  
   ![45](images/SS12.jpg)

   3.コードを書く  
   ![46](images/SS13.jpg)

   4.Keybit8をPCに接続してからTool → PortでKeybit8のポートを選ぶ。  
   どれがKeybit8のポートか分からない場合は、一度ケーブルを抜き、再度挿した時に新しく増えた番号を選択してください。  
   ポートが表示されない場合はKeybit8のリセットスイッチを2回素早く押すと表示されます。   
   多くの場合、Port番号の末尾に「Arduino Leonardo」と記載があります。  
   ![47](images/SS14.jpg)  

   5.画面左上のマイコンボードに書き込むボタン（赤枠のボタン）を押して、Keybit8のリセットスイッチを「カチカチッ」と素早く2回押します。  
   ![48](images/SS15.jpg)  
   画面右下に「Done uploading」と表示されれば成功です。



   - Arduino IDEの入門としてLEDをチカチカさせるコードを2通り記載します。ぜひお試しください。
      ```
      // LED を1秒ごとにチカチカさせる

      int ledPin = 9;  // 緑LEDのピン番号

      void setup() {
        pinMode(ledPin, OUTPUT);  // ピンを出力に設定
      }

      void loop() {
        digitalWrite(ledPin, HIGH);  // 点灯
        delay(1000);                 // 1秒待つ
        digitalWrite(ledPin, LOW);   // 消灯
        delay(1000);                 // 1秒待つ
      }
      // delay(1000) の数字を変えると点滅の速さが変わるので試してみてください。
      ```
      
      ```
      // キーを押すとLEDがチカチカする

      // キースイッチマトリクス（2×4）
      const int ROW[2] = {21, 20};
      const int COL[4] = {4, 5, 6, 7};

      // キー対応LED 9個　LED[0]～LED[7]が赤、LED[9]が緑
      const int LED[9] = {1, 0, 8, 16, 14, 15, 18, 19, 9};

      // 圧電スピーカー
      const int buzzerPin = 10;

      void setup() {
        pinMode(LED[0], OUTPUT);  // LED1を出力に設定
        pinMode(ROW[0], OUTPUT);  // ROW1を出力に設定
        pinMode(COL[0], INPUT_PULLUP);  // COL1を入力に設定
        digitalWrite(ROW[0], LOW);  //ROW1をLOW(GND)にする
      }

      void loop() {
        digitalWrite(LED[0], HIGH); // LED1点灯
        delay(500); //500ミリ秒待ち
        digitalWrite(LED[0], LOW); // LED1消灯
        if(digitalRead(COL[0]) == LOW){
          delay(500); //もしSW1が押されていれば500mS待ち
        }
      }
      ```
